Connect your Database
One-time setup — takes about 5 minutes
1
Create a free Supabase account
Go to supabase.com → Sign up → New project. Pick any name and region.
2
Run this SQL to create the tables
In your project go to SQL Editor and run:
create table users ( id bigint generated always as identity primary key, name text not null, pin text not null, email text default '', role text default 'staff', color text default '#5b6af5' ); create table requests ( id bigint generated always as identity primary key, user_id bigint references users(id) on delete cascade, type text not null, start_date date not null, end_date date not null, note text default '', status text default 'pending', submitted timestamptz default now() ); alter table users enable row level security; alter table requests enable row level security; create policy "open" on users for all using (true) with check (true); create policy "open" on requests for all using (true) with check (true);
3
Paste your Project URL and anon key below
Find these under Project Settings → API in your Supabase dashboard.
